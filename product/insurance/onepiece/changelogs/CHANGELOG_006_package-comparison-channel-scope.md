# Changelog 006: Package Comparison Channel Scope

> **Date:** 2026-03-05
> **Layer:** Capability (Quotation & Comparison)
> **Status:** Documented

---

## What Changed

Documented that the package comparison view is currently available only in the online sale channel. Branch staff currently selects packages without a comparison view.

Added planned state: package comparison will be available across all sale channels.

## Documents Updated

- `PRODUCT.md` — Capability Registry entry for Quotation & Comparison updated with current/planned channel note
- `capabilities/quotation/CAPABILITY.md` — Added business rules QT-009 (current: branch has no comparison) and QT-010 (planned: comparison in all channels)

## Rationale

The comparison view is a key differentiator of the platform. Restricting it to online-only limits branch staff effectiveness. The plan is to extend comparison to all sale channels to provide a unified experience.
