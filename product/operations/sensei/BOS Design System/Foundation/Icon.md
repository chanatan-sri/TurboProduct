## Usage in HTML

```html
<!-- Reference icon as SVG symbol -->
<svg class="bos-icon" width="24" height="24">
  <use href="#Search" />
</svg>

<!-- Or as an img / inline SVG from Figma export -->
<img src="icons/Search.svg" alt="Search" width="24" height="24" />
```

```css
.bos-icon {
  width: 24px;
  height: 24px;
  display: inline-block;
  vertical-align: middle;
  fill: currentColor;
}

.bos-icon--sm {
  width: 16px;
  height: 16px;
}
```

---

## Usage Notes

- Standard icons are **24×24px**. Product/Insurance icons are **16×16px**.
- Use **outline** icons by default; switch to **filled** for active/selected/alert states.
- Always provide an `alt` text or `aria-label` for accessibility.
